# SafeMed Agenda

## Descrição do projeto
O SafeMed Agenda é um sistema desenvolvido com a proposta de auxiliar clínicas de pequeno porte no controle de atendimentos, organização da agenda e gerenciamento de pacientes e profissionais.

## Objetivo
O projeto tem como objetivo oferecer uma solução simples e funcional para o agendamento, remarcação e cancelamento de consultas, além da consulta de horários disponíveis e do cadastro de pacientes e profissionais.

## Funcionalidades previstas
- Cadastro de pacientes
- Cadastro de profissionais
- Agendamento de consultas
- Remarcação de consultas
- Cancelamento de consultas
- Consulta de agenda
- Visualização de horários disponíveis
- Pesquisa de pacientes

## Público-alvo
Clínicas de pequeno porte que precisam organizar melhor sua rotina de atendimento.

## Tecnologias previstas
- Python
- Interface gráfica
- Banco de dados simples

## Status do projeto
Projeto acadêmico em fase inicial de estruturação.
